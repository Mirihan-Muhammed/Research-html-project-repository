What is Programming language

Programming language is a set of instruction and other syntax use to create a software program. Languages that programmers use to write code are called "high-level languages." This code may be compiled into a "low-level language," which is recognized directly by the computer hardware. Examples of high-level languages are C++, Java, Perl, and PHP. Languages as C++ and Java are being called "compiled languages" as the source code must first be compiled in order to run.
